Description du projet

Présentation du projet

Le projet consiste en une calculatrice simple développée en Python, capable de réaliser les opérations de base : addition, soustraction, multiplication et division.

L’objectif est d’apprendre les notions de fonctions, conditions et gestion des erreurs en Python.

Caractéristiques principales :
  • Interface en ligne de commande
  • Fonctions dédiées pour chaque opération
  • Gestion des erreurs (division par zéro, saisie incorrecte)
  • Expérience utilisateur améliorée (messages clairs)
Ce projet m’a permis de renforcer mes bases en Python et de comprendre la logique des programmes interactifs.
Description du projet

Présentation du projet

Le projet consiste en un site web pour l'Hôtel Chambord, un établissement fictif, avec une interface moderne et responsive.

L'objectif est de présenter les chambres, les services, et de permettre aux visiteurs de réserver en ligne.

Caractéristiques principales :
  • Site web responsive adapté aux mobiles et tablettes
  • Pages : Accueil, Chambres, Services, Réservation, Contact
  • Formulaire de réservation fonctionnel avec validation des données
  • Galerie d’images pour les espaces communs
  • Navigation claire et expérience utilisateur optimisée
Ce projet m’a permis de renforcer mes compétences en HTML, CSS, JavaScript et en design web interactif.
